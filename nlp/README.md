## NLP是什么？ 

NLP = NLU(Natural Language Understanding) + NLG(Natural Language Generation)

NLP为什么难？

- 表达同一种意思可以有多种不同的方式/句子
- Ambiguity：一词多义 - 学习数据，上下文处理

## 自然语言处理四个维度

- Semantic 语义
- Syntax 句子结构 （句法分析、依存分析）
- Morphology 单词 (分词、词性）
- Phonetics 声音
